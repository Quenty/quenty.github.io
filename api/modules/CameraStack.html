CameraStack

Back to reference

Holds camera states and allows for the last camera state to be retrieved.

Also initializes an impulse and default camera as the bottom of the stack. Is a singleton.

Functions

Functions Description
PrintCameraStack() Outputs the camera stack
GetDefaultCamera() Returns the default camera
GetImpulseCamera() Returns the impulse camera.
GetRawDefaultCamera() Returns the default camera without any impulse cameras
GetTopState() Retrieves the top state off the stack
GetNewStateBelow() Returns a new camera state that retrieves the state below its set state
GetIndex(State) Retrieves the index of a state
Remove(State) Removes the state from the stack
Add(State) Adds a state to the stack
