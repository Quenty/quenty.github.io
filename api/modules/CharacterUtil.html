GetHumanoid (Descendant)
Retrieves a humanomid from a descendant (Players only).

Parameters:

  • Descendant The child you're searching up from. Really, this is for weapon scripts.

Returns:

    A humanoid in the parent structure if it can find it. Intended to be used in

     workspace  only. Useful for weapon scripts, and all that, especially to work on non
 player targets. Will scan *up* to workspace . If workspace   has a humanoid in it, it
 won't find it.

    Will work even if there are non-humanoid objects named "Humanoid" However, only works on objects named "Humanoid" (this is intentional)
